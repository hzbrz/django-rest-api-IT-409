Customers

*created using generic ListCreateAPIView so get/post in same view

Get Customer(s):

Gets a JSON list of all customers

GET/POST: localhost:8000/listCustomerREST/

Add Customer:

Creates a customer by taking in a json object

GET/POST: localhost:8000/listCustomerREST/

Addresses

Get Addresses:

Gets a JSON list of all addresses related to a customer id

GET: localhost:8000/getAddressREST/customer_id/

- by default customer_id is set to 1. Create customers and change it to test/get other customers' addresses

Add Address:

Creates an address by taking in a json object

POST: localhost:8000/addAddressREST/

Orders

Get Orders:

Gets a JSON list of all orders related to a customer id

GET: localhost:8000/getOrderREST/customer_id/

- by default customer_id is set to 1. Create customers and change it to test/get other customers' orders

Add Order:

Creates an order by taking in a json object. Some fields (order_date, order_total, expiration_date)
are hidden because they either default to a value or change based on other calculations.

POST: localhost:8000/addOrderREST/

Order Items

Get Products:

Gets a JSON list of all products (order items) related to an order id

GET: localhost:8000/getProductREST/order_id/

- by default order_id is set to 1. Create orders and change it to test/get other order's items

Add Product:

Creates a product by taking in a json object. The quantity determines the order total for the associated order.

POST: localhost:8000/addProductREST/
